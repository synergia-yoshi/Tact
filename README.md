# Tact Prototype

Tact is a Netlify-ready GTM execution prototype for turning sales strategy into repeatable AI-assisted daily workflows.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Netlify

Netlify can deploy this repository with:

- Build command: `npm run build`
- Publish directory: `dist`

The same settings are defined in `netlify.toml`.
